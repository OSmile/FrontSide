Условия. Продолжение

Попробуем создавать значения флагов динамически:
Помимо написания значений флагов самому, их можно получать. Самый распространённый способ – операции сравнения. Подробнее прочитать можно тут: RAVESLI
Также, существуют операции с флагами: И(flag1 && flag2), ИЛИ(flag1 || flag2), НЕ(!flag). Их комбинации могут создавать новые флаги, которые можно использовать.
Чтобы использовать флаги, необязательно вводить переменные. Например:
int a = 0;
int b = 5;
if (a == 0)
{
  std::cout « “На нуль делить нельзя!» « std::endl;
}
else
{
  b = b / a;
}
Мы могли ввести новую переменную для сохранения результата сравнения, но плохой тон создавать переменные для одноразовых действий.
Рассмотрим случай, когда условие состоит из трёх состояний, например, больше-меньше-равно.
int a = 5;
if (a > 0)
{
  /*Обработка первого условия/*
}
else
{
  if (a < 0)
  {
    /*Обработка условия*/
  }
  else
  {
    /*Обработка условия*/
  }
}
В данном случае: Если значение переменной больше 0 – выполнится первый блок кода; Если меньше нуля – второй.
Поскольку мы поставили строгие условия, остался третий вариант, не удовлетворяющий первым двум – значение переменной равно 0, выполнится третий блок кода.
Заметьте по структуре, что два блока последних блока являются вложенными.
