Переменные

Отлично, конечно, работать только с постоянными значениями, но в реальности такого не будет никогда, для этого данные нужно хранить и анализировать. Основная работа в программировании осуществляется с переменными. Они имеют собственный тип, имя, размер, и значение. Более подробно предлагаю почитать тут: METANIT или TUTORIALSPOINT
Изложим кратко:

  • int - Тип для хранения целочисленных переменных. Примеры: int intValue = 354;;
  • float, double - Типы для хранения вещественных чисел. Примеры: float floatValue = -324.0876;;
  • char - Тип для хранения символьных данных. Пример: char charValue = ‘h’;;
  • string - Тип для хранения и работы со строковыми переменными. Доступно при подключении директив iostream и string.h.
Также не забывай, что из одной переменной может быть получена другая переменная. Пример: int a = b + c;
Давайте доработаем программу из лекции 2:
#include <iostream>
#include <string>

int main()
{
  std::string stringValue = "Hello, world!";
  std::cout « stringValue;
  return 0;
}
