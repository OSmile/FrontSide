Лекция 1

Строение начального файла

Надеюсь, ты готов? Давай начнём!
В данной лекции, приоткроем для тебя завесу, скрывающую работу с кодом, и начнём со строения файла MainSource.cpp:
#include <iostream>
int main ()
{
  return 0;
}
 Расскажу чуть более подробно о файле:

  • 1. #include <iostream> - интегрированная в систему библиотека; используемая в обыденной работе с языком;
  • 2. int main() - функция, на текущий момент, единственная в проекте. В реальных работающих проектах, их количестве будет много больше;
  • 3. Далее идёт тело функции. В последующих лекциях, мы расскажем более подробно о том, что может быть в рамках { } .
