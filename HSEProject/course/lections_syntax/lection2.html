Переменные

Для хранения данных используются переменные.
Существует несколько типов переменных.
Переменные для хранения чисел:
int - целочисленные переменные;
float, double - вещественные числа;
Подробный разбор числовых переменных и областей их допустимых значений можно найти тут: tutorialspoint
Для хранения символов существует тип данных char.
Для работы со строками - string.
Давайте доработаем программу из задания 1:
#include <iostream>
#include <string>
int main() {
  std::string string = "Hello world!";
  std::cout « string « std::endl;
  return 0;
}
